# Rust で始める Web API 開発 - 実践ガイド

このガイドは、Rust を使用してモダンで安全な Web API を開発する方法を、実践的な例を通じて学べます。

## 想定読者

- Rust の基礎知識がある方（変数、関数、if/match などの基本構文）
- Web API 開発経験がある方（HTTP、REST などの概念を理解している）
- パフォーマンスと堅牢性を重視するアプリケーション開発に興味のある方

## 本の構成

### イントロダクション（第1章）

このガイドのテーマ、学習目標、必要な環境構築について説明します。

### 環境構築（第2章）

Rust 開発環境の整備から、Axum プロジェクトの初期設定までを進めていきます。

### 最初の API を作る（第3章）

簡単なユーザー管理 API を実装しながら、Axum の基本的な使い方を習得します。

## 学習成果

このガイドを読了すれば、以下ができるようになります：

- Axum で HTTP ハンドラーを実装できる
- [[async-await]] を使った非同期プログラミングができる
- [[result]] 型によるエラーハンドリングができる
- JSON のシリアライズ・デシリアライズができる

## 習得時間

全章を通して 2-3 時間で学習できます。
