# 所有権（Ownership）

Rustでは、すべての値は所有者を持ちます。所有権とは、値がメモリ上のどこに存在し、いつ解放されるのかを自動的に管理する仕組みです。

## 所有権のルール

所有権には3つの基本的なルールがあります：

1. Rustの各値は、オーナー（owner）という所有者を持つ
2. 値は同時に1つのオーナーのみを持つ
3. オーナーがスコープを抜ける時、値は削除される

## コード例

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1;  // 所有権がs1からs2に移動
    // println!("{}", s1);  // エラー：s1は所有権を失った
    println!("{}", s2);    // OK
}
```

:::message[move でコピーを避ける]{tip}
所有権の移動（move）は、メモリのコピーを避けるための工夫です。大きなデータ構造でも効率的に値を受け渡せます。
:::

関連する概念として、[[borrowing]]と[[lifetime]]があります。
