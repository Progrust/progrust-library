# 辞書タグ一覧

辞書コンテンツで使用するタグの一覧（執筆規約）。システムによる検証は行われない（content-model R-18: タグは自由記述）ため、この一覧を運用で正とする。

## 運用ルール

- 辞書エントリのタグは必ずこの一覧から選ぶ（1エントリ1〜3個目安）
- 一覧に適切なタグがない場合は、**先にこの表へ行を追加**してから使う。追加時は既存タグ・「吸収する近縁表記」列と照らし、意味が重なるタグを作らない
- 「吸収する近縁表記」列は表記ゆれの防止用。この列にある語をタグとして新設しない（該当タグを使う）

## タグ一覧

| タグ           | 説明                                            | 吸収する近縁表記                     |
| -------------- | ----------------------------------------------- | ------------------------------------ |
| 基本文法       | 変数・関数・制御構文などの入門項目              | 文法, 構文, 入門                     |
| 所有権         | 所有権・ムーブ・Copy                            | ownership, ムーブ, move              |
| 借用           | 参照・借用規則・ライフタイム                    | 参照, lifetime, ライフタイム         |
| 型システム     | 型推論・ジェネリクス・型変換                    | 型, ジェネリクス, generics           |
| プリミティブ型 | bool・数値型・char・strなど言語組み込みの基本型（The Rust Reference準拠） | primitive, primitive type, スカラー型 |
| トレイト       | trait定義・実装・トレイトオブジェクト           | trait, インターフェース              |
| エラー処理     | Result / Option / panic                         | Result, Option, 例外                 |
| パターンマッチ | match式・let束縛のパターン                      | match, パターン                      |
| コレクション   | Vec / HashMap などのコレクション型              | Vec, HashMap, 配列                   |
| イテレータ     | Iterator・アダプタ・遅延評価                    | iterator, 反復処理                   |
| 文字列         | String / &str・文字列操作                       | String, str                          |
| 並行処理       | スレッド・Send / Sync・チャネル                 | マルチスレッド, concurrency, 並列    |
| 非同期         | async / await・Future・非同期ランタイム         | async, tokio                         |
| メモリ         | スマートポインタ・ヒープ/スタック・Box/Rc/Arc   | スマートポインタ, Box, Rc            |
| マクロ         | 宣言的マクロ・手続き的マクロ                    | macro, derive                        |
| モジュール     | mod / use・クレート構成・可視性                 | クレート, use, パッケージ            |
| ツールチェーン | cargo・rustup・rustfmt / clippy などの開発ツール | cargo, rustup, ビルド               |
